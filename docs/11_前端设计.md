# 前端设计

# 1. 文档目标

定义前端系统架构。

技术栈：

```text
React 19
Vite
TypeScript
TailwindCSS
ReactFlow
TanStack Query
Zustand
Shadcn/UI
```

---

系统定位：

```text
AI Research Agent Platform
```

而非：

```text
普通聊天机器人
```

---

# 2. 设计目标

前端需要展示：

```text
任务

Workflow

Agent执行过程

工具调用

记忆系统

研究报告
```

---

而不仅是：

```text
输入
↓
输出
```

---
