# 记忆系统设计

---

# 1. 文档目标

本文档定义系统长期记忆（Long-Term Memory）设计方案。

本系统采用：

```text
ContextAgent
+
MemoryAgent
```

双层设计。

其中：

```text
ContextAgent
```

负责：

```text
记忆召回（Recall）
```

而：

```text
MemoryAgent
```

负责：

```text
记忆更新（Update）
```

---

# 2. 设计目标

本系统的记忆并非聊天记忆（Chat Memory）。

而是：

```text
Research Memory
```

即：

研究记忆系统。

系统需要长期积累：

- 历史研究结果
- 趋势快照
- 结构化洞察
- 主题演化过程

用于未来研究任务。

---

# 3. 设计原则

## 原则一：记忆与状态分离

运行时状态：

```text
AgentState
```

---

长期记忆：

```text
Qdrant
```

---

Workflow结束后：

```text
AgentState
```

可销毁。

长期记忆必须保留。

---

## 原则二：记忆服务统一访问

禁止：

```python
QdrantClient()
```

出现在：

```text
Agent
Workflow
Tool
```

中。

---

所有记忆操作必须通过：

```python
MemoryService
```

完成。

---

## 原则三：可追溯

所有记忆必须记录：

```text
来源

时间

证据
```

禁止保存无法验证的结论。

---

## 原则四：支持时间演化分析

系统必须支持：

```text
2026-07 MCP

VS

2026-08 MCP

VS

2026-09 MCP
```

趋势比较。

---

# 4. 系统架构

```text
                ┌─────────────┐
                │ ContextAgent│
                └──────┬──────┘
                       │
                       ▼
                 MemoryService
                       │
                       ▼
                     Qdrant


Workflow
    │
    ▼

MemoryAgent
    │
    ▼
MemoryService
    │
    ▼
Qdrant
```

---

职责划分：

| 模块          | 职责     |
| ------------- | -------- |
| ContextAgent  | Recall   |
| MemoryAgent   | Update   |
| MemoryService | 统一接口 |
| Qdrant        | 存储     |

---

# 5. 记忆分类

系统维护三类长期记忆。

---

## ResearchMemory

研究记忆。

保存：

完整研究结果。

---

示例：

```text
MCP发展趋势研究

研究时间：
2026-08-01
```

---

作用：

```text
历史研究复用
```

---

## TrendSnapshot

趋势快照。

保存：

某个时间点的趋势状态。

---

示例：

```text
MCP

GitHub项目数：132

论文数：45

模型数：28
```

---

作用：

```text
趋势比较
```

---

## InsightMemory

洞察记忆。

保存：

经过证据验证的高价值洞察。

---

示例：

```text
MCP正在成为Agent生态标准协议
```

---

作用：

```text
跨任务知识复用
```

---

# 6. 基础记忆模型

---

## MemoryRecord

统一记忆基类。

```python
class MemoryRecord(BaseModel):

    memory_id: str

    memory_type: str

    topic: str

    summary: str

    created_at: datetime

    source_ids: list[str]

    metadata: dict = {}
```

---

字段说明：

| 字段        | 说明     |
| ----------- | -------- |
| memory_id   | 唯一ID   |
| memory_type | 类型     |
| topic       | 主题     |
| summary     | 摘要     |
| source_ids  | 来源     |
| metadata    | 扩展信息 |

---

# 7. ResearchMemory

```python
class ResearchMemory(
    MemoryRecord
):
```

# 17. 去重机制

写入前执行：

```python

```

---

```python
0.95
```

```text
视为重复记忆
```

不写入。

# 18. 记忆污染控制
